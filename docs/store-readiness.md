## Technical Compliance

### Permissions
- Web: No special permissions (file picker only via `<input type="file">`)
- Android (Capacitor): Storage read/write for downloads
- iOS: File access via native picker

### Data Deletion
- Users can delete individual transfers from History page
- Files auto-expire after 24 hours
- Full account deletion via support request (7-day window)

### Security
- All transfers encrypted in transit (TLS)
- Files encrypted at rest (Supabase Storage)
- Claim codes are 8-char alphanumeric (1.5T combinations)
- Share codes are UUID-based
- MCP tokens hashed with SHA-256

## Accessibility

### WCAG 2.1 AA Compliance
- Color contrast: All text meets 4.5:1 minimum ratio
- Focus indicators: `ring-2 ring-accent` on interactive elements
- Touch targets: minimum 44×44px (`min-h-[44px]`)
- Screen reader: Semantic HTML, aria labels where needed
- Keyboard navigation: All actions accessible via keyboard
- No auto-play, no flashing content
